# 🚀 Project 01 - Embeddings

## 📌 Objective

Understand how AI models convert text into numerical vectors (embeddings) and compare semantic similarity between sentences.

---

## 📚 Concepts Covered

- Sentence Embeddings
- Sentence Transformers
- NumPy Arrays
- Embedding Dimensions
- Cosine Similarity
- Semantic Search Basics

---

## 🛠 Technologies Used

- Python
- PyTorch
- Sentence Transformers
- NumPy
- Scikit-Learn
